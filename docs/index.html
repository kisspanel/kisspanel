KissPanel Installer

Log in to your server e.g. ssh root@your.server then download the installation script:

wget https://raw.githubusercontent.com/kisspanel/kisspanel/release/install/kisspanel-install.sh

Check you are running as the root user, configure the options you want below, then run:

bash kisspanel-install.sh

System Configuration

Must be a number between 2000-9999
Select interface language
Must be a valid hostname format (e.g., server.domain.com)
Must be a valid email address
Must start with a letter and contain at least one uppercase letter, one lowercase letter, and one number
Enable disk quota management for user accounts
Enable API access for automation
Enable interactive installation process
Force installation even if system requirements are not met
